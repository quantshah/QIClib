Quantum Information and Computation library (QIC_lib)

A C++11 library for quantum information and computation based on Armadillo

Version 0.0.1 - November 16, 2015

QIC_lib is a mordern C++11 library for general purpose quantum computing, supporting Linux, Windows and Mac OS X. It is a header only template library, using Armadillo (developed by Conrad Sanderson et al., Data61, Australia) for highly efficient linear algebra calculations, and if available, the NLopt nonlinear optimization library for certain features.
